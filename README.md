# 🌸 Online Flower Delivery Website (HTML, CSS, JavaScript)

A beautifully designed **flower delivery website** built using **HTML**, **CSS**, and **JavaScript**. This is a fully responsive **frontend project** for an **online florist** or **gift shop** concept, showcasing modern **UI/UX** principles.

## 🛠️ Features
- Elegant homepage with flower categories and banner
- Product listings with prices and "Add to Cart" buttons
- Contact and About sections
- Responsive for mobile, tablet, and desktop
- Clean and semantic HTML5 & CSS3 structure

## 🎨 Technologies Used
- HTML5 for page structure
- CSS3 for styling and layout
- JavaScript for interactivity (cart, toggle nav, etc.)
